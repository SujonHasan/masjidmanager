# Roles

- `super_admin`: SaaS owner across all mosques.
- `owner`: Mosque account owner.
- `admin`: Mosque admin with full operational access.
- `cashier`: Can manage financial entries and receipts.
- `imam`: Can read salary/profile data and publish allowed content later.
- `member`: Can read their own payment history later.
- `guest`: Read-only public/transparency access later.
